**Iridium ADC** Bill Blakely (#) Abstract Modern digital audio systems are pushing performance boundaries. Most high quality digital to analog converters for audio are capable of 24 bit 96 kHz conversion. Professional audio interfaces are usually capable of 24 or 32 bit, 192 kHz sample rates. These systems are able to achieve a dynamic ranges greater than 100 dB, and distortion of just a few parts per million. This project proposes a high resolution analog digital converter (ADC) to measure and characterize these types of devices. The system will also be useful anywhere high dynamic range measurements are desired. The digitized data will be transferred over USB so the system can be used with existing software such MATLAB, custom python scripts, or ARTA labs. (#) Goals/Deliverables (##) Minimum 1. 24-bit, 384 kSPS converter. 4. Single channel. 4. Data transfer over USB. 5. Precise, accurate voltage reference. 2. SNR > 100 dB 3. THD < 0.01 % (##) Desired 1. 768 kSPS conversion 4. Two channels of conversion. 5. Switchable input impedance. 6. Range selector. 2. SNR > 110 dB 3.
